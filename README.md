# 🍾 漂流瓶 Drift Bottle

為 ADD/ADHD 設計的療癒記憶與待辦 App

## ✨ 功能

- 🍾 **漂流瓶** — 零摩擦待辦事項
- 🎙️ **語音記錄** — 說話自動建立待辦
- 📋 **日常清單** — 出門前確認、睡前儀式等
- ✦ **靈感珍珠** — 靈感視覺化，從沙粒養成璀璨珍珠
- 🪼 **放鬆區** — 水母呼吸、戳泡泡、環境音樂
- 🌅 **日夜主題** — 根據時間自動切換四種氛圍

## 🚀 部署到 GitHub Pages

1. 建立新 repo
2. 推送所有檔案到 `main` branch
3. Settings → Pages → Source: `main`, `/ (root)` → Save
4. 等一兩分鐘即可使用

## 📱 安裝到手機

在手機瀏覽器打開後，點「加入主畫面」即可安裝為 App。
